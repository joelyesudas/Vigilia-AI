# 📖 Overview

Vigilia AI is an **AI-powered desktop Security Operations Center (SOC)** that continuously monitors systems, detects cyber threats, analyzes security events, maps attacks to the **MITRE ATT&CK Framework**, and provides real-time security insights through an interactive dashboard.

Designed for **Critical National Infrastructure (CNI)**, Vigilia AI helps security teams identify threats early, assess organizational risk, and improve cyber resilience using intelligent automation.

---

# 🚨 Problem Statement

Critical infrastructure organizations generate thousands of security events every day.

Traditional monitoring solutions are often:

- Expensive
- Complex to deploy
- Cloud-dependent
- Difficult for small organizations

Security analysts spend valuable time manually reviewing logs instead of responding to incidents.

---

# 💡 Our Solution

Vigilia AI provides an intelligent desktop-based SOC that:

- Detects suspicious activities using AI
- Continuously monitors endpoints
- Performs scheduled security scans
- Calculates security risk
- Maps attacks to MITRE ATT&CK
- Delivers real-time alerts
- Visualizes security posture through an interactive dashboard

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/joelyesudas/Vigilia-AI.git
```

Go to the project

```bash
cd Vigilia-AI
```

Install dependencies

```bash
npm install
```

Install frontend dependencies

```bash
cd frontend
npm install
```

Run the application

```bash
npm run dev
```
